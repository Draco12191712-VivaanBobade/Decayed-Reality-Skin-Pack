# Contributing

Thank you for your interest in contributing to this Minecraft Bedrock skin pack!

## Before You Start

Please:

- Read the README.
- Read the Code of Conduct.
- Check existing Issues and Pull Requests before creating new ones.

## Types of Contributions

We welcome:

- New Translations
- Skin improvements
- Bug fixes
- Documentation updates
- Localization improvements
- Repository maintenance
- Version Compatibility Maintenance

## Skin Requirements

All submitted skins must:

- Be original work or properly licensed.
- Follow Minecraft Bedrock skin specifications.
- Use PNG format.
- Have a transparent background where appropriate.
- Avoid offensive, hateful, or inappropriate content.
- Be tested in Minecraft Bedrock Edition.

## File Naming

Use clear and descriptive filenames:

Good examples:

- aiden.png
- discodude.png
- sheldon.png

Avoid:

- skin1.png
- finalfinal.png
- newskin.png

## Folder Structure

Place files in the appropriate directories:

```text
root/
├── aiden.png
├── discodude.png
├── sheldon.png
```


## Creating a Pull Request

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test the skin in Minecraft Bedrock Edition.
5. Commit your changes with a clear commit message.
6. Open a Pull Request.

## Pull Request Guidelines

Include:

- Description of the change
- Screenshots if applicable
- Testing information

Example:

- Added a new mcsm skin.
- Tested on Minecraft Bedrock 26.x.
- Verified skin geometry and textures display correctly.

## Commit Message Examples

```text
Fix Translation error
Fix transparency on Aiden skin
Update contributor documentation
```

## Reporting Issues

When reporting an issue, please include:

- Minecraft version
- Platform (Windows, Android, iOS, Xbox, etc.)
- Description of the problem
- Screenshots if available

## Licensing

By contributing to this repository, you agree that your contributions may be distributed under the project's license.
