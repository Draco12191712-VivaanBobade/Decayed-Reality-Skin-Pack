# Security Policy

## Supported Versions

The following versions of the skin pack currently receive support:

| Version | Supported |
|----------|----------|
| Latest Release | ✅ |
| Previous Release | ⚠️ Limited |
| Older Releases | ❌ |

## Reporting a Security Issue

Although this project is a Minecraft Bedrock skin pack and does not contain servers, executables, or account systems, we still appreciate reports regarding:

- Malicious or suspicious files
- Copyright violations
- Unauthorized content
- Malicious links in documentation
- Repository abuse
- Packaging issues that could negatively affect users

Please do not create a public Issue for sensitive reports.

Instead:

1. Contact me (the maintainer) directly if possible.
2. Provide a detailed description of the issue.
3. Include screenshots, links, or reproduction steps if applicable.

## What to Expect

After a report is received:

- The issue will be reviewed.
- Additional information may be requested.
- Appropriate action will be taken if necessary.
- A fix or removal may be included in a future release.

## Scope

This policy applies to:

- Skin textures
- Pack manifests
- Repository documentation
- Downloaded release packages
- Community contributions

This policy does not apply to:

- Minecraft Bedrock Edition itself
- Third-party websites
- External software not maintained by this project

## Responsible Disclosure

Please report concerns privately whenever possible and allow maintainers reasonable time to investigate before publicly disclosing details.

Thank you for helping keep this project safe and trustworthy for everyone.
